  • 교사는 여전히 악성 민원에 직접 노출

AI 모델 1 — 학교 데이터 통합 검색

TF-IDF (한국어 char n-gram 2~4) + cosine similarity · 66건 인덱싱

학교 FAQ

학교 공지
15
가정통신문

  • 출처 라벨: 📖 FAQ / 📢 공지 / 📨 가정통신문 / 📋 매뉴얼 명시 — 학부모 신뢰
  • 강매칭 (관련도 ≥0.30) → FAQ 답변 그대로 표시 (빠름·무료)
  • 약/무매칭 → Claude Haiku로 자연어 답변 생성 (AI 모델 2)
  • + NEIS Open API 실시간 호출 (학사일정·급식)로 학교 단위 맞춤
정식 도입 시: 시도교육청 공지·학교 사이트 RAG 자동 인덱싱 (전국 1.2만 학교)

AI 모델 2 — Claude Haiku 4.5 자연어 답변

FAQ 약·무매칭 시 LLM이 컨텍스트 기반 자연어 답변 생성

입출력 흐름

  • 입력: 학부모 질문 + 학교 FAQ Top 3 컨텍스트
  • 출력: 친절·간결한 한국어 답변 + 다음 행동 안내
  • 비용: 호출당 약 $0.001 (Haiku 4.5, in 1000 / out 400 토큰)
  • 캐시: st.cache_data — 같은 질문 무료 재호출

시스템 프롬프트 원칙

  • 교사의 정당한 교육활동·생활지도·평가 존중
  • 무리한 요구는 정중히 안내 (강요·즉시 처벌 표현 금지)
  • 3~5문장 간결 답변 + 다음 행동 명확
  • 근거 법령·매뉴얼 한 줄 인용
  • 모르면 모른다고 정직하게 (담임/행정실/교육청 안내)
💬 예시 "선생님 책상에 꽃 한 송이 놓아도 될까요?" → 청탁금지법 안내 + 공식 채널 권장

AI 모델 3 — 민원 분류기 + 유사 사례 RAG

TF-IDF + cosine · 학습 37건 (정당 22 + 모호 3 + 악성 12)

분류 카테고리 13종

  • 정당 안전 / 학사·평가 / 건강 배려 / 시설·환경 / 정보 요청
  • 모호 대화 권장 / 정보 부족
  • 악성 교권침해 / 업무 부담 / 반복·집요 / 사적 영역

위험도 점수 + 보정 로직

  • 0.0 ~ 1.0 (강매칭 우선)
  • 약매칭 (<0.30) → 위험도 ×0.4 (false positive 방지)
  • 정당 강매칭 (≥0.35) → ×0.2 (정당 신호)
  • 유사 과거 사례 Top 3 + 판정·근거 함께 표시
정식 도입 시: 교육부 교권보호위 의결문 수천 건으로 미세조정 → 정확도 ↑

AI 모델 4 — 위험도 평가 + 학교용 분석 리포트

위험도 + 유사 사례 + 규칙 기반 마크다운 리포트 자동 생성

위험도임계권장 학교 대응 절차
🚫 악성 위험≥ 0.65학교장·교감 우선 검토 · 교권보호위 회부 사전 협의 · 교사 단독 응대 금지, 복수 입회
⚠️ 주의≥ 0.451차 담임 + 보호자 면담 (학교 내 공식 공간) · 면담 기록 보존 · 필요 시 상담교사 동석
✅ 정당< 0.45관할 담당자(담임/보건/시설 등) 정상 처리 · 처리 결과 회신 · 매뉴얼 업데이트
📋 학교용 마크다운 리포트: 분류 결과 + 민원 원문 + 유사 과거 사례 + 권장 절차 자동 생성
  • → 교사 직접 응대 부담 ↓ + 학교장 사전 검토 가능 + 표준 대응 자동 안내

Claude가 학부모 질문 1,005개 생성 → AI 분류 → 정확도 측정 → 프롬프트 자가 진화

🔁 AI 자가개선 루프 (사람 개입 X)

  • 분류기가 100건 자체 테스트 → 오분류 패턴 추출
  • Claude가 분석 결과 보고 시스템 프롬프트 자동 개선
  • 새 프롬프트로 재테스트 → 정확도 비교
  • 종료: +3%p 미만 또는 3회 도달 또는 90%+ 달성
  • 실제 결과: 악성 recall +29.3%p 자가 보정 성공

⚠️ 발견된 한계 (정직 명시)

  • 악성 recall ↑ 시 모호 recall ↓ Trade-off 발견
  • 단순 프롬프트 변경으로는 다목표 동시 최적화 불가
  • 정식 도입 시 다음 보강 필요:
    • 교육부 교권보호위 의결문 수천 건 fine-tuning
    • 강화학습(RLHF) 또는 다목표 최적화
    • 휴먼 검증자 인터페이스
💡 AI가 자기 성능을 측정하고 스스로 개선 — 시도·성공·한계까지 정직 명시 (논리성·합리성 ↑)
