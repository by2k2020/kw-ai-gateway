PROBLEM STATEMENT
왜 지금 — AI 게이트웨이
2023 교사 사망 사건 이후 교사들은 학부모 악성 민원에 무방비. 입수 단계 AI 필터링은 부재.
  • 2023.7 교사 사망 사건 — 학부모 악성 민원 사회 의제화
  • 2024.3 교권보호5법 시행 — 교사 보호 의무 강화
  • 2025 학교민원 응답시스템 추진 — 진행 중
  • 그러나 민원 입수 단계의 AI 필터링은 부재
  • 교사는 여전히 악성 민원에 직접 노출
  • 학부모도 "답을 주는 도구" 부재

def smart_answer(query): # RAG: 60 사례 전체 컨텍스트 ctx = load_complaint_samples() resp = haiku.create( system=SYSTEM_PROMPT, messages=[ctx + query], max_tokens=800 ) return resp.json() # {answer, category, risk_score, reasoning}

SYSTEM PROMPT 원칙

  • 교사 정당 교육활동 존중 — 무리한 요구 정중 안내
  • 비공식 표현 가이드 (기 죽인다 / 차갑다 / 차별 등)

  • 모르면 정직하게 — 담임·행정실 안내
