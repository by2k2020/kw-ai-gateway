# 학교 민원 AI 게이트웨이

> **제8회 교육 공공데이터 AI활용대회** · AI 활용 아이디어 기획 (일반 부문) 출품작

학부모-학교 사이에 **AI 게이트웨이**를 두어 민원이 학교에 도달하기 전 분류·검색·평가·요약하는 시스템. 학부모는 자가 점검 후 신중하게 결정, 학교는 분석 리포트 기반 차분 대응 → **교사 보호 + 학부모 만족 + 교권 침해 사전 예방**.

## 핵심 발견

- **2024 보호자 가해 비중 10.9%** — 2022년 6.7%에서 급증
- **유치원 100%, 초등 30%** 가 보호자 가해 → 우리 시스템 핵심 타겟층
- 보호자 가해 1위 = **"정당 교육활동 부당간섭" 20%** → 우리 게이트웨이가 직접 막을 수 있는 유형
- 출처: 교육부 2024학년도 교육활동 침해 실태조사 (2025.5.13)

## 시스템 구조 (5단계)

```
[학부모] 민원 작성
   ↓
[AI 게이트웨이]
  1. 민원 분류 (정당/모호/악성 위험)
  2. 유사 사례 RAG 검색
  3. 자가 점검 결과 표시 (~70% 자체 종료 가설)
  4. (계속) 서면 양식 자동 가이드
  5. 학교용 분석 리포트 자동 생성
   ↓
[학교] 리포트 기반 차분한 대응
```

## AI 모델

| # | 모델 | 기술 |
|---|---|---|
| 1 | 민원 분류기 | TF-IDF (한국어 char n-gram 2~4) + cosine similarity |
| 2 | 유사 사례 RAG 검색 | 벡터 공간 cosine top-k |
| 3 | 위험도 평가 + 리포트 생성 | 가중 합산 + 룰 기반 템플릿 |
| 4 | 시도별 5년 시계열 예측 | Holt-Winters 감쇠 추세 |
